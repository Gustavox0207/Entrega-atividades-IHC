## UA1 – Introdução à IHC e seus benefícios

### Objetivo

Propor melhorias de IHC para uma caixa de diálogo de exclusão de arquivos, focando em clareza e prevenção de erros.

### Solução Desenvolvida

A atividade consistiu em identificar falhas de usabilidade em uma interface de confirmação de exclusão e sugerir alterações nos rótulos dos botões ("Apagar" e "Cancelar"), cores para destaque visual e títulos mais informativos para evitar ações acidentais.

### Arquivos Entregues

- desafio1.txt

## UA2 – Interface, interação e Affordance

### Objetivo

Analisar a acessibilidade de interfaces para usuários com deficiência visual e propor soluções de hardware e software.

### Solução Desenvolvida

Foi discutida a importância de interfaces otimizadas para leitores de tela e comandos de voz, além de listar equipamentos essenciais como teclados Braille e fones de ouvido para garantir a inclusão e eficiência na navegação.

## UA4 – TypeScript

### Objetivo

Implementar classes e lógica de negócios utilizando TypeScript para gerenciar produtos e vendas.

### Funcionalidades Implementadas

- Criação da classe `Produto` com atributos específicos.
- Implementação da classe `Venda` para gerenciar coleções de produtos.
- Método para cálculo automático do valor total da venda.

### Tecnologias Utilizadas

- TypeScript
- Node.js
- ts-node

### Como Executar o Projeto

#### Pré-requisitos

- Node.js instalado
- npm ou yarn

#### Instalação

```bash
npm install
```

#### Execução

```bash
npm start
```

#### Compilação

```bash
npm run build
```

### Arquivos Entregues

- index.ts
- package.json
